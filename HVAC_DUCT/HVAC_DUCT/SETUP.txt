HVAC_DUCT - Simple Setup
========================

QUICK SETUP (2 steps):
1. Open AutoCAD Architecture
2. Type: APPLOAD
3. Browse to: D:\PROJECT_API_CAD\MECH\HVAC_DUCT\HVAC_DUCT\HVAC_DUCT\
4. Select: HVAC.lsp
5. Click: Load
6. Done!

AUTOMATIC SETUP (Startup Suite):
1. Type: APPLOAD
2. Click: Contents...
3. Click: Add...
4. Select: HVAC.lsp
5. Click: Open
6. Click: Close
7. Restart AutoCAD

USAGE:
- Ticks load automatically when opening files
- Draw polylines and use TANDUCTGOOD command
- Deleted objects are automatically cleaned up

COMMANDS:
- TANDUCTGOOD: Draw ticks on polyline
- AUTO: Load ticks manually
- CLEANUPTICKS: Clean up deleted objects
- REGISTERERASE: Enable auto-cleanup

FILES:
- HVAC.lsp: Main LISP file
- HVAC_DUCT.dll: Compiled DLL (x64)
- HVAC_DUCT_Ticks_[FileName].txt: Tick data files
